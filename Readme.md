# Breast Cancer Classification using Machine Learning

## Project Overview

This project predicts whether a breast cancer tumor is **Malignant** or **Benign** using Machine Learning algorithms.

The project demonstrates the complete ML workflow including:
- Data Loading
- Data Exploration
- Data Preprocessing
- Feature Scaling
- Model Training
- Model Evaluation
- Data Visualization



## Dataset

The project uses the **Breast Cancer Wisconsin Dataset** available in Scikit-learn.

Target Classes:
- Malignant
- Benign



## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn



## Machine Learning Models

### Logistic Regression
- Standard Logistic Regression
- Balanced Logistic Regression (`class_weight='balanced'`)


## Evaluation Metrics

The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve
- ROC-AUC


## Visualizations

The project includes:

- Feature Distribution Histograms
- Correlation Heatmap
- ROC Curve
- Dataset Analysis
